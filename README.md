API for personal music list
===========================

Kostya Overchenko

.Net 6 <br>
EntityFrameworkCore <br>
MS_Sql<br>

API allows you to create artists and songs, to get a list of all artists and their songs. 

Run Command:

  cd MyMusic.Api
  dotnet run
